# Bella Rose Admin Panel - Guide simple d'utilisation

## 1) Objectif du Bella Rose Admin Panel

Bella Rose Admin Panel est l'espace d'administration du site.  
Il sert a modifier le contenu sans toucher au code :

- ajouter / modifier / supprimer des images (carrousel, medias),
- publier des articles,
- changer les textes de la page d'accueil,
- mettre a jour les contacts (WhatsApp, email, telephone).

Chaque changement enregistre dans Bella Rose Admin Panel apparait ensuite sur le site.

## 2) Connexion

1. Ouvrir le site vitrine.
2. Cliquer sur **Administration**.
3. Entrer email + mot de passe.
4. Vous arrivez sur le tableau de bord Bella Rose Admin Panel.

## 3) Comprendre les zones de l'interface (les "coins")

### Barre laterale gauche

C'est le menu principal :

- **Photos et fichiers (Media)** : toutes les images du site.
- **Galerie d'apprentissage / Carrousel** : images du carrousel.
- **Articles (Conseils)** : blog du site.
- **Infos generales du site** : logo, nom, contacts, textes accueil.
- **Services**, **Temoignages**, etc.

### Zone centrale

C'est la liste des elements de la section choisie (ex: liste des images, liste des articles).

### Haut de page

- **Creer** : ajouter un nouvel element.
- **Rechercher / Filtres** : retrouver un element.
- **Selection multiple** : cocher plusieurs lignes pour supprimer en lot.

### Colonne / panneau de droite (sur une fiche)

C'est l'endroit ou on remplit les champs : titre, image, description, etc.

### Bouton Sauvegarder

Toujours cliquer sur **Sauvegarder** apres modification.

### Bouton "Voir le site web" (retour rapide)

Ce bouton sert a revenir rapidement sur le site public pour verifier les changements.

Ou le trouver exactement :

1. Regarder la **barre laterale gauche** (menu principal).
2. Descendre **tout en bas de ce menu**.
3. Vous verrez un bouton/lien : **Voir le site web**.

Si vous ne le voyez pas :

- faites defiler la barre laterale vers le bas;
- agrandissez la fenetre du navigateur;
- verifiez que vous etes bien connectee dans Bella Rose Admin Panel.

## 4) Cas concret : gerer le carrousel (ajouter / supprimer)

### Ajouter une image au carrousel

1. Aller dans la section **Galerie d'apprentissage / Carrousel**.
2. Cliquer sur **Creer**.
3. Remplir les champs demandes (titre/caption si present).
4. Choisir une image.
5. Cliquer **Sauvegarder**.

Resultat : l'image apparait sur le site.

### Supprimer une image du carrousel

1. Aller dans la meme section.
2. Cocher la ligne de l'image.
3. Cliquer **Supprimer**.
4. Confirmer.

Resultat : l'image disparait du site.

## 5) Regles importantes pour eviter les erreurs

- Toujours cliquer sur **Sauvegarder**.
- Attendre 2 a 5 secondes puis actualiser le site.
- Ne pas supprimer un element si on n'est pas sure.
- Utiliser des titres clairs (ex: "Salon bleu 01").
- Pour les images : privilegier JPG/PNG/WebP de bonne qualite.

## 6) Sections les plus utiles au quotidien

### Infos generales du site

Modifier contacts + textes de la page d'accueil.

### Articles (Conseils)

Ajouter les nouveaux articles de blog.

### Carrousel / Galerie d'apprentissage

Mettre les nouvelles images visibles sur la page.

### Photos et fichiers

Bibliotheque globale des images.

## 7) Verification apres chaque modification

1. Sauvegarder dans Bella Rose Admin Panel.
2. Aller dans la barre laterale gauche.
3. Cliquer sur **Voir le site web** (en bas du menu).
4. Controler la page concernee sur le site public.
5. Si besoin : revenir dans Bella Rose Admin Panel, corriger, sauvegarder.
