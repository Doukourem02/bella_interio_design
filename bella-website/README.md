# 🏠 Luxe Home - Premium Furniture Website

A modern, responsive furniture e-commerce website built with Next.js, featuring elegant animations, clean design, and premium user experience.

## ✨ Features

### 🎨 Modern UI/UX Design
- **Clean & Professional** - Minimalist design with elegant typography
- **Responsive Layout** - Optimized for all devices (desktop, tablet, mobile)
- **Smooth Animations** - Professional hover effects and transitions
- **Consistent Branding** - Cohesive color scheme and visual elements

### 🧩 Component Architecture
- **Modular Components** - Reusable and maintainable code structure
- **Header Navigation** - Animated menu with left-to-right sliding bars
- **Product Collections** - Interactive product cards with hover effects
- **Testimonials Slider** - Auto-playing customer reviews carousel
- **Newsletter Signup** - Email subscription with background imagery
- **Footer Links** - Comprehensive site navigation with animations

### 🚀 Interactive Features
- **Animated Menu Bars** - Elegant left-to-right sliding animations on hover
- **Product Showcase** - Professional product cards with ratings and pricing
- **Customer Reviews** - Auto-rotating testimonials with navigation controls
- **Email Subscription** - Newsletter signup with form validation
- **Social Media Integration** - Connected social media profiles
- **Mobile-Responsive Menu** - Slide-out mobile navigation

## 🛠️ Tech Stack

- **Framework**: [Next.js 14](https://nextjs.org/) - React framework for production
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) - Utility-first CSS framework
- **Icons**: [Lucide React](https://lucide.dev/) - Beautiful & consistent icons
- **UI Components**: [shadcn/ui](https://ui.shadcn.com/) - Radix UI components
- **Animations**: CSS transitions and transforms
- **Typography**: Custom font optimization with Next.js

## 🚀 Getting Started

### Prerequisites
- Node.js 18+ 
- npm, yarn, pnpm, or bun

### Installation

1. **Clone the repository**
```bash
git clone https://github.com/waleedcodes/Luxe-Home-Furniture-Website.git
cd furniture-website
```

2. **Install dependencies**
```bash
npm install
# or
yarn install
# or
pnpm install
```

3. **Run the development server**
```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

4. **Open your browser**
Navigate to [http://localhost:3000](http://localhost:3000) to see the website.
